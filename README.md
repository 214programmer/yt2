# Channel Scope

Веб-приложение для Vercel, которое по ссылке на YouTube-канал собирает метрики через `YouTube Data API`, а затем делает глубокий AI-аудит через `Groq`.

Что умеет:

- анализирует канал и доступные публичные видео;
- объясняет, почему самое популярное видео стало популярным;
- показывает преимущества и недочеты ведения канала;
- дает практические советы и план роста;
- подбирает конкурентные видео, которые залетели в похожей тематике.

## 1. Куда вставить API ключи

Создайте файл `.env.local` в корне проекта и вставьте туда свои ключи:

```env
YOUTUBE_API_KEY=ваш_ключ_youtube_api
GROQ_API_KEY=ваш_ключ_groq
GROQ_MODEL=openai/gpt-oss-20b
```

Где брать ключи:

- `YOUTUBE_API_KEY`: Google Cloud Console -> включите `YouTube Data API v3` -> создайте API key.
- `GROQ_API_KEY`: [Groq Console](https://console.groq.com/keys).

`GROQ_MODEL` можно не менять. По умолчанию используется `openai/gpt-oss-20b`.

## 2. Как запустить локально

```bash
npm install
npm run check:api
npm run dev
```

После запуска откройте:

```text
http://localhost:3000
```

`npm run check:api` делает мини-проверку ключей: один запрос к YouTube API и один structured-запрос к Groq.

## 3. Как задеплоить на Vercel

1. Загрузите проект в GitHub.
2. Импортируйте репозиторий в Vercel.
3. В `Project Settings -> Environment Variables` добавьте:
   - `YOUTUBE_API_KEY`
   - `GROQ_API_KEY`
   - `GROQ_MODEL`
4. Запустите deploy.

## 4. Коды доступа

Коды для запуска анализа лежат в `lib/access-codes.js`.

Администратор может добавлять и удалять строки в массиве `ACCESS_CODES`. Все коды равнозначны. База данных и регистрация не используются.

## 5. Технические заметки

- Серверный маршрут: `app/api/analyze/route.js`
- YouTube-сборщик: `lib/youtube.js`
- AI-анализ через Groq: `lib/groq.js`
- Интерфейс: `app/page.jsx`
- Коды доступа к запуску анализа: `lib/access-codes.js`

## 6. Ограничения

- Для очень больших каналов приложение анализирует до `60` последних видео, чтобы не упираться в лимиты API и время выполнения на Vercel.
- Точность конкурентной выборки зависит от того, насколько хорошо тема канала считывается по названиям и метаданным.
- Groq делает выводы только по тем данным, которые были получены из YouTube API.
